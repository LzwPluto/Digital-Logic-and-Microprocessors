❓ 帮助

逻辑模拟器使用方法：

1. 创建电路：

- 左侧工具栏添加逻辑门、模块、IO设备。

- 拖动移动组件，点击连接点创建连线。

2. 编辑电路：

- 选择模式：可选中多个组件移动。

- 删除元件：双击组件即可删除。

- 位编辑器：点击寄存器等位直接修改。

- 替换元件：选中后点击目标元件替换。

3. 仿真控制：

- 时钟控制：调节速度、暂停/继续。

- 竞态模式：开关影响全局，请谨慎操作。

4. 存档与分享：

- 保存：导出电路为 JSON 文件。

- 读取：加载本地 JSON 文件。

- 创意工坊：浏览社区分享存档。

5. 导出 Verilog：

- 导出代码仅在 ModelSim 测试通过，其他仿真器可能不兼容。
